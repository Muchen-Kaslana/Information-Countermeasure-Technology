# 实验9：木马基本功能的实现

## 内容概述
本实验中，通过设置好源、目的 IP 地址，在客户端发送指令到服务端，实现木马的基本功能。

### 功能列表
1. 输出字符串：在服务端输出指定字符串。
2. 关机：使服务端主机在60秒内关机。
3. 取消关机：在关机计时内取消服务端主机的关机操作。
4. 获取C盘文件列表：获取服务端主机C盘的文件列表。
5. 截屏：捕获服务端主机的桌面图像。
6. 删除：从服务端主机的C盘中删除指定文件。
7. 上传：从客户端主机选择文件，并传送到服务端主机并保存为"myFile.txt"。
8. 下载：从服务端主机选择文件，并将其内容拷贝到客户端主机保存为"myFile.txt"。
